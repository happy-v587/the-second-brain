# the-second-brain

你的第二个大脑，在复杂的信息流中发现更真实的你。

```
+-------+     +---------+     +--------+
| 信息流 | --> | 第二大脑 | --> |决策&执行|
+-------+     +---------+     +--------+ 
```

信息流包括：基于微信公众号、书籍内容、知乎博文、小红书博文、今日头条、抖音、快手、B站、淘宝、京东... 的点赞、收藏
第二大脑：这是一个知识中心，将信息流的数据记录
决策&执行：有了大量的信息后，ai可以根据类别总结出你的爱好、风格、todo list，最终帮助你更好的生活
